Привет, Мир!

Этот сайт написан в primo.af — бесплатном генераторе статичных сайтов, совмещающим в себе IDE и CMS.

Тут можно писать в колонках.

  • Делать списки

  • В общем типа есть маркдаун

Но самое главное, можно добавлять компоненты, в которых можно без кода менять параметры (как вкладка Контент в Тильде)

То есть при желании можно собрать свой конструктор сайтов и страниц для портфолио. Он публикуется на гитхаб пейджес, а с помощью Vercel или Netlify можно всё это прокинуть на свой домен.
